📋 PRÓXIMAS AÇÕES (Manual no Xcode):

1. Abrir: open ios/Runner.xcworkspace

2. Adicionar SPM Firebase SDKs
   File → Add Packages
   https://github.com/firebase/firebase-ios-sdk
   Selecionar: FirebaseAnalytics, FirebaseMessaging, FirebaseCrashlytics

3. Adicionar Push Notifications Capability
   Runner → Signing & Capabilities → + Capability → Push Notifications

4. Build & Test
   flutter build ios --debug
   flutter run -d iPhone
