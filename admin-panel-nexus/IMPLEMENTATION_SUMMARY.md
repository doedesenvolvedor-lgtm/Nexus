## 🚀 Como Usar

### Instalação Rápida
```bash
cd /workspaces/Nexus/admin-panel-nexus
npm install
npm run dev
```

Acesse: `http://localhost:3000`

**Credenciais Demo:**
- Email: `admin@nexus.com`
- Senha: `admin123456`

### Build para Produção
```bash
npm run build
npm run preview
```
